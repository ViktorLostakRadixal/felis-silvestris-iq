How the Test Works

After entering the Experiment ID and pressing the "Start" button, the application will switch to fullscreen mode. Pulsating circles will appear on the screen. The goal is for the kitten to touch them.

The test will run for the specified duration. If the duration is left empty, the test will run indefinitely until you manually stop it by pressing the Escape key. All interactions are recorded.
